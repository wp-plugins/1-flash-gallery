== Description ==

Are you looking for a best way to manage and display photo gallery on your blog? Try "1 Flash Gallery" - photo gallery plugin with slideshow function. It provides a comprehensive interface for managing photos and images through a set of admin pages, and it displays image gallery in a way that makes your web site look very nice. You can display galleries with a beautiful image gallery skins integrated with "1 Flash Gallery".

For more information review [photo gallery demos](http://1plugin.com "Flash Gallery Wordpress Plugin Demo"), created with "1 Flash Gallery" Wordpress Plugin

If you have question review [FAQ](http://1plugin.com/faq "Photo Gallery FAQ") section


Features:

* [Flipping Book with Page Flip effect](http://1plugin.com/page-flip "Page Flip")
* Image gallery full screen viewing
* Slideshow function
* Multiple albums/categories per post
* Configurable photo gallery background, logo and highlight color
* Sort images feature
* Cool flash skin
* [Widget](http://1plugin.com/faq/95 "1 Flash Gallery Widget")
* Numerous image arrays may be controlled from a solitary shell. It is very simple to alter the kind of image array with a tap on the mouse. 
* Potent administrative panel.  It is possible to simply activate any application within the image array or slide show together with a World press plug-in via the administrative panel.
* Stylish animatronics function within the image array. Your photo array will be awe-inspiring and slideshows  will be meaningful and radiant.
* It is possible to  upload photographs to your image array in various fashion.: via Local Network, FTP transfer, ZIP file, URL, or even via objects from a Word press Gallery or other blogging centers.
